## Getting Started in Three Winks:

1. **Install CatalystGo CLI:**
   ```bash
   go install github.com/catalystgo/cli
   ```

2. **Create Your Microservice:**
   ```bash
   catalystgo create my-microservice
   cd my-microservice
   ```

3. **Run Your Microservice:**
   ```bash
   catalystgo run
   ```
